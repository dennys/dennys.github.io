Use IFTTT to send currency exchange rate mail daily

Requirement

I want to have a daily mail including currency exchange rate information.

  1. Mail subject: include the exchange rate in the mail subject.
  2. Mail content:
    1. Has some links to banks’ exchange rate page.
    2. Include some exchange rate history chart.

Solution

Use ifttt.com

  1. Create a new Applet on IFTTT
  2. Add a new “If This”

  1. Choose “finance” service

  1. Choose “Today’s exchange rate report”

  1. Choose the input/output currency and trigger time.

  1. Add a new “Then That”

  1. Choose a “gmail” service

  1. If you just want to send yourself an email, you can click the right button. If you want to send to several recipients, you can click left button.

  1. This is an example to send your self an email.
