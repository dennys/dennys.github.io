Use IFTTT to send currency exchange rate mail daily

Dennys published on 2022-03-24 included in IT
Requirement I want to have a daily mail including currency exchange rate information. Mail subject: include the exchange rate in the mail subject. Mail content: Has some links to banks’ exchange rate page. Include some exchange rate history chart. Solution Use ifttt.com Create a new Applet on IFTTT Add a new “If This” Choose “finance” service Choose “Today’s exchange rate report” Choose the input/output currency and trigger time. Add a new “Then That” Choose a “gmail” service If you just want to send yourself an email, you can click the right button.

Procedure You can reference the official document: https://checkmarx.atlassian.net/wiki/spaces/SD/pages/169246832/SonarQube+Plugin+v8.5.0+and+up Download the plugin from here, it only supports SonarQube LTS version (for now, it’s 8.x) Configurea Quality Gate/Profiles of SonarQube for Checkmarx’s rules.
