更新 Unlock 過的 Pixel 5

Dennys published on 2022-06-02 included in IT
事前準備 下載 Pixel OTA Images Pixel Factory Images 把redfin-sq1a.220205.002-factory-734556ab.zip (檔案名稱看實際的版本) 解壓縮, 要解兩層, 直到解出 boot.img 利用 Magisk 修補 boot.img, 產生的檔案名稱長類似這樣 magisk_patched-24201_LSQ4E.img, 再把這個檔案傳回電腦中 進入開發者模式並確定 adb devices 可以連 更新 執行以下指令重新開機進入sideload mode adb reboot sideload 刷入安全性更新檔案, 這個步驟會比較久 adb sideload redfin-ota-sq1a.220205.002-84ba2421.zip 完成上傳後勿重新開機，“在手機上” 選擇Reboot to bootloader 刷入被 Magisk 修補後的 boot.img fastboot flash boot –slot=all magisk_patched-24201_LSQ4E.img 執行以下指令重新開機 fastboot reboot 進入 設定->關於->Android 版本, 確定有更新成功. 參考 https://forum.xda-developers.com/t/4356219/ https://www.mobile01.com/topicdetail.php?f=565&t=6487485

無聲遠端桌面連線 - VNC

Dennys published on 2022-06-02 included in IT software
需求 要做一個 步驟 (Server 端) 把 show accept/reject prompt for each connection 關掉, 這樣連線上去時就不會跳一個視窗出來問是否同意連線. 關掉 Notify when users connect and disconnect, 這樣連線時就不會跳出視窗. 另外, 預設是如果一段時間沒動作, 就會自動斷線. 如果不希望這樣, 可以把 Disconnect users who have been idle for an extended period 取消, 這樣就算放一整天也不會斷線. (也可以在 Expert 裡面設定 idleTimeout 時間) 如果沒有關掉, 那有人連上來時, 就會跳出這個視窗, 就被發現了. :p 步驟 (Client 端) 在連線以前, 把 Option 裡面的 View-only 打開, 這樣你連上去的時候, 就不會去動到對方的滑鼠. 成果 依照以上設定好, 就可以神不知鬼不覺得連到對方電腦了. 當然, 對方如果進去 VNC 看, 還是可以看到有誰連進來, 這個就沒招了. 如果臨時想去移動對方的鍵盤滑鼠怎麼辦? 很簡單, VNC 的工具列可以改設定, 直接把 View-only 拿掉就可以了.

Dennys published on 2022-06-01 included in IT software
Asuswrt-Merlin 提供兩種 VPN server 工具, PPTP 和 OpenVPN. PPTP 看起來已經被淘汰, 連 iOS 10 以上都不支援 PPTP 了. 因此目前看來是只能用 OpenVPN. 註1: Asuswrt 官方原版似乎有提供 PPTP, OpenVPN, IPSec VPN 這三種 VPN server, 可參考 https://www.asus.com/tw/support/FAQ/1044190/, 不過我已經刷 Asuswrt-Merlin 了, 就不測試了. 在 Asuswrt-Merlin 設定 OpenVPN 進入 VPN 伺服器, 他可以建立兩個 OpenVPN server, 預設是 Server 1, 把它開啟. 我是只有要透過 VPN 來連內網, 所以設定 “LAN only”. 如果需要使用者輸入帳號密碼, 請在最後面新增一個 username. 下拉是選單這裡改選 “進階設定” 進階設定如下, 如果希望 user 在 login 的時候要輸入帳號密碼, 那 “Username/Password Authentication 就需要選 “是”. 設定好後, 請再回到一般設定, 然後把設定值輸出到成一個 .
