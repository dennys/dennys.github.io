# Google Pixel 5 Unlock / Root / Magisk  **注意: 系統刷機有一定的風險, 可能導致手機無法開機和失去保固, 資料還會被清空, 執行前請先三思!!!** **注意: 系統刷機有一定的風險, 可能導致手機無法開機和失去保固, 資料還會被清空, 執行前請先三思!!!** **注意: 系統刷機有一定的風險, 可能導致手機無法開機和失去保固, 資料還會被清空, 執行前請先三思!!!** \=================================================== 安裝 Google USB driver -------------------- 1. 首先要在電腦上安裝 Google USB driver, 可在這裡下載[https://developer.android.com/studio/run/win-usb](https://developer.android.com/studio/run/win-usb) 2\. 解開壓縮檔 (應該是 usb\_driver\_r13-windows.zip), 在 android\_winusb.inf 按右鍵, 然後一直往下就好了 [![](https://dennys.files.wordpress.com/2022/02/image-11.png?w=485)](https://dennys.files.wordpress.com/2022/02/image-11.png) ps: 沒有裝 driver 的話, 在執行 fastboot 的時候, 就會遇到 waiting for any device 的訊息 [![](https://dennys.files.wordpress.com/2022/02/image-12.png?w=317)](https://dennys.files.wordpress.com/2022/02/image-12.png) \=================================================== 安裝 Android SDK Platform Tools ----------------------------- 從這裡下載, 一直 next next 就好. https://developer.android.com/studio/releases/platform-tools#downloads \=================================================== 手機解鎖 Unlock ----------- **請記得先備份資料, 因為解鎖會清空資料** **請記得先備份資料, 因為解鎖會清空資料** **請記得先備份資料, 因為解鎖會清空資料** 1\. 手機進入開發者模式並確定 adb devices 可以連上手機 [![](https://dennys.files.wordpress.com/2022/01/auth.jpg?w=197)](https://dennys.files.wordpress.com/2022/01/auth.jpg) 2\. Enable OEM 解鎖 [![](https://dennys.files.wordpress.com/2022/01/image-13.png?w=934)](https://dennys.files.wordpress.com/2022/01/image-13.png) 3\. 執行下列指令, 手機會出現要你確認是否要 unlock, 按下去, 手機就解鎖了. **注意: 這裡會把手機資料清空** ``` adb reboot bootloader fastboot flashing unlock ``` [![](https://dennys.files.wordpress.com/2022/01/image-12.png?w=405)](https://dennys.files.wordpress.com/2022/01/image-12.png) \=============== 以上完成 Unlock, 以下為 Root 的步驟 =============== 手機 Root ------- 1\. 用手機下載並安裝 Magisk (https://github.com/topjohnwu/magisk\_files/blob/canary/app-debug.apk) 2.1 檢查手機版本 (設定 -> 關於, 然後拉到最下面), 譬如我這支是 SQ1A.220105.002 [![](https://dennys.files.wordpress.com/2022/02/image-2.png?w=235)](https://dennys.files.wordpress.com/2022/02/image-2.png) 2.2 用電腦連到 https://developers.google.com/android/images, 下載對應的 boot image (按 Link) [![](https://dennys.files.wordpress.com/2022/01/image-17.png?w=848)](https://dennys.files.wordpress.com/2022/01/image-17.png) 2.3 解開裡面的 vbmega.img 和 boot.img. [![](https://dennys.files.wordpress.com/2022/01/image-16.png?w=497)](https://dennys.files.wordpress.com/2022/01/image-16.png) 3\. 將前一步驟解出來的 boot.img 放到手機上, 然後使用 Magisk 對 boot.img 做 patch, 並把 patch 完後的檔案傳回電腦 [![](https://dennys.files.wordpress.com/2022/02/image.png?w=295)](https://dennys.files.wordpress.com/2022/02/image.png) [![](https://dennys.files.wordpress.com/2022/02/image-1.png?w=291)](https://dennys.files.wordpress.com/2022/02/image-1.png) 4\. 將手機重開, 並同時按下 電源+音量下鍵進入 bootloader 模式 5.1 用 fastboot 刷 vbmega.img, 不然刷完後重開機, 手機會出現 failed to load/verify boot images (應該是 Pixel 4A 5G 以後的都需要這個) ``` fastboot flash --disable-verity --disable-verification vbmeta vbmeta.img ``` [![](https://dennys.files.wordpress.com/2022/01/image-15.png?w=760)](https://dennys.files.wordpress.com/2022/01/image-15.png) 5.2 用 fastboot 刷剛剛用 Magisk 給 patch 過的 image. 注意, 這裡用的是 fastboot boot, 不是 fastboot flash. 刷完以後會自動重開機. ``` fastboot boot magisk_patched.img ``` [![](https://dennys.files.wordpress.com/2022/01/image-14.png?w=824)](https://dennys.files.wordpress.com/2022/01/image-14.png) Magisk ------ 6\. 使用 Magisk 檢查是否 Root 若你的 Masgisk 長這樣 [![](https://dennys.files.wordpress.com/2022/02/image-6.png?w=59)](https://dennys.files.wordpress.com/2022/02/image-6.png) 然後執行時出現這個畫面, 這時候按確定, 他應該會去嘗試下載, 然後最後說剖析套件時發生問題. [![](https://dennys.files.wordpress.com/2022/02/image-7.png?w=242)](https://dennys.files.wordpress.com/2022/02/image-7.png) 還是得裝前面講的版本 (https://github.com/topjohnwu/magisk\_files/blob/canary/app-debug.apk), 執行後看到下面畫面, 就是 Root 成功了. [![](https://dennys.files.wordpress.com/2022/02/image-3.png?w=289)](https://dennys.files.wordpress.com/2022/02/image-3.png) 7\. Root 之後可再刷第三方 ROM, Pixel5 有一堆可刷, 可參考 https://forum.xda-developers.com/f/11343/. 比較意外的是, 本來以為最多人用的是 [LineageOS](https://www.google.com/search?q=LineageOS), 不過看來 [ProtonASOP](https://www.google.com/search?q=ProtonASOP) 受到的關注更多 (以 XDA 上面的 views/replies 數量來看). 還有 [CleanSlate](https://forum.xda-developers.com/t/4192641/) 在沒有 Root 的情況下也可以使用, 這個值得研究一下.
